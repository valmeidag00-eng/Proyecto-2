# Proyecto 2 

## Descripción
Proyecto del Sprint 2 del Bootcamp de QA. Se enfoca en el **diseño de casos de prueba manuales** para funcionalidades web, como formularios y navegación, y el uso de herramientas de gestión como Jira.

### Objetivo
Aprender a diseñar casos de prueba estructurados y a usar herramientas de gestión de bugs para documentar resultados.

---

## Actividades principales

1. **Técnicas de diseño de casos de prueba**
   - Clases de equivalencia
   - Valores límite
   - Tablas de decisión

2. **Formato de casos de prueba**
   - Condiciones previas
   - Pasos a ejecutar
   - Datos de prueba
   - Resultado esperado

3. **Gestión de bugs con Jira**
   - Creación de tickets
   - Adjuntar capturas de pantalla
   - Seguimiento básico de bugs

4. **Pruebas adicionales**
   - Responsividad en distintos dispositivos
   - Compatibilidad básica entre navegadores

---

## Logros

- Diseño de **más de 30 casos de prueba** para formularios y navegación web.
- Primeros reportes de bugs documentados en Jira.
- Comprensión de técnicas de diseño y formatos profesionales de casos de prueba.
